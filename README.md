# YTDL
```js
const ytdls = require('ytdls')
ytdls.ytv('link yt')
.then(res => console.log(res))
.catch(err => console.log(err))
```
and
```js
const ytdls = require('ytdls')
ytdls.yta('link yt')
.then(res => console.log(res))
.catch(err => console.log(err))
```
installation
```js
npm install ytdls
```
